# 🎾 TENNIS BETTING CALCULATOR - DATA REQUESTS

## How to Get Match Data

Instead of storing multiple markdown files, simply request data for any specific match you need.

### Request Format

Just provide:
1. **Tournament**: e.g., "WTA Brisbane", "ATP Challenger Nottingham"
2. **Players**: e.g., "Sabalenka vs Muchova", "Monday vs Chidekh"

### Example Requests

```
"fetch data for wta-brisbane aryna sabalenka vs karolina muchova"
"fetch data for atp-challenger-nottingham monday vs chidekh"
"get stats for wawrinka vs bergs"
```

The system will:
- ✅ Fetch H2H records
- ✅ Get current rankings & stats
- ✅ Calculate betting odds
- ✅ Generate calculator input values
- ✅ Provide match predictions
